# フロントエンド

ToDo管理のフロントエンドを、次の順で実装していきます。

## 目次

### [ページ外観の作成](todo-page/README.md)

ToDoページの外観を作成します。

### [コンポーネントの分割](component/README.md)

ToDoページをコンポーネントに分割します。

### [REST APIクライアントの作成](api/README.md)

REST APIクライアントコードを使用する準備として、クライアントコードをラッピングしたオブジェクトを作成します。

### [CORSの設定](cors/README.md)

バックエンドのREST APIにアクセスできるようにするため、CORSを設定します。

### [モックサーバの起動](mock/README.md)

OpenAPIドキュメントを使用してモックサーバを起動し、REST APIの呼び出しをテストできるようにします。

### [ToDoの一覧表示](todo-list/README.md)

ToDoページで登録されているToDoを一覧表示できるように実装します。

### [ToDoの登録](todo-add/README.md)

ToDoページで新しいToDoを登録できるように実装します。

### [ToDo状態の更新](todo-update/README.md)

ToDoページでToDoの状態を更新できるように実装します。

### [ToDo一覧の絞り込み](todo-filter/README.md)

ToDoページで表示されるToDoを絞り込みできるように実装します。
